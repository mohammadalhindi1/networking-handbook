# Packet Captures

This directory will contain small, sanitized captures used by the handbook.

Before committing a capture:

- capture only traffic required for the lesson;
- remove credentials, tokens, cookies, personal addresses, and unrelated traffic;
- use documentation networks where possible;
- record the capture environment, command/filter, protocol, purpose, and expected packet sequence;
- include a text walkthrough so learners can follow the example without downloading a binary file.
